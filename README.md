# PaperRPG Project Structure

This document outlines the structure of the PaperRPG project.

## Directory Layout

*   **`PaperRPG-Website/`**: This directory contains the website source code for PaperRPG. It was moved inside this project directory to consolidate all related resources. **AI Agents: Please note that website-related tasks should be performed within this directory.**
*   **`paper_rpg/`**: Core logic or backend modules.
*   **`electron/`**: Electron-specific main process code.
*   **`src/`**: React/Frontend source code for the main application.
*   **`public/`**: Static assets.

## Notes for AI Agents
*   The `PaperRPG-Website` folder is a sub-project located within this root. When asked to modify the website, verify if the user means this sub-directory.
